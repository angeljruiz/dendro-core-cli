# Usage
<!-- usage -->
```sh-session
$ npm install -g dendro-cli
$ dendro COMMAND
running command...
$ dendro (-v|--version|version)
dendro-cli/0.0.0 linux-x64 node-v14.15.3
$ dendro --help [COMMAND]
USAGE
  $ dendro COMMAND
...
```
